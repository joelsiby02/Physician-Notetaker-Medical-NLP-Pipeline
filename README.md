##  Features

### 1. Medical Named Entity Recognition (NER)
Extracts key clinical elements:

- Symptoms & complaints
- Diagnoses (rule-prioritized)
- Treatments / medications / procedures
- Temporal expressions (dates, times, durations, counts)

### 2. Structured Medical Summary (JSON)
Produces a clean, schema-compliant report containing:

- Accident / visit context
- Chief complaint & symptoms
- Diagnosis
- Treatment provided / recommended
- Current clinical status
- Prognosis
- Functional limitations
- Physical examination findings

### 3. Keyword Extraction
Uses **KeyBERT** to identify the most representative medical phrases and concepts.

##  To run this project : -

### 1. Environment setup

```bash
# Create and activate environment
conda create -n physician-notetaker python=3.12 -y
conda activate physician-notetaker

# Install dependencies
pip install -r requirements.txt
```

### 2. Run the pipeline

1. Place your transcript in  
   `data/sample_transcript.txt`

2. Execute:

```bash
python run_pipeline.py
```

All results are saved in the `outputs/` folder.
